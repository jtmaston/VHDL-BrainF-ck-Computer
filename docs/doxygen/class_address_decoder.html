AddressDecoder Entity Reference

Detailed Description

Address decoder entity. This effectively acts as a shift register: a value loaded onto addressLines is transformed into its corresponding chip select (CS) lines, onto @outputAddressLines, to be used with RAM/ROM cells inside its associated block.

Usage example: AddressDecoder_tb.vhd

Example wavefile:

Precondition
Address must be loaded on addressLines before the rising edge of clock
Postcondition
Address lines can be considered valid @falling_edge of clock
